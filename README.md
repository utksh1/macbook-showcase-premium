# Premium 3D MacBook Scroll Showcase

An ultra-premium, visually stunning, and immersive 3D MacBook showcase featuring realistic scroll animations, built with **Three.js**, **GSAP**, and **Lenis**.

## 🚀 Live Demo
**[macbook-3d.vercel.app](https://macbook-3d.vercel.app)**

## ✨ Key Features
- **Realistic 3D Model**: High-fidelity MacBook Pro model with physical materials and detailed textures.
- **Cinematic Scroll Animations**: Smooth, frame-by-frame 3D transitions synchronized with user scrolling.
- **Premium Dark Aesthetics**: Unified dark unibody design with refined reflective properties.
- **Minimalist Dash Navigation**: A custom 7-dash vertical indicator for intuitive section tracking.
- **Responsive & Performance Optimized**: Optimized assets and smooth scrolling for a premium feel on all devices.

## 🛠️ Tech Stack
- **Three.js**: 3D rendering and material management.
- **GSAP (GreenSock)**: Orchestrating the cinematic scroll-based animations.
- **Lenis**: Smooth scrolling engine.
- **Vite & TypeScript**: Modern frontend development environment.

## 📂 Project Structure
- `/src/main.ts`: Core logic for Three.js scene setup and GSAP animations.
- `/src/style.css`: Premium layout and UI component styling.
- `/public/models/`: Contains the optimized 3D MacBook model.
- `/public/media/`: High-quality UI previews and video assets.
